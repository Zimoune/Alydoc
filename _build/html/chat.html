Le chat¶

Il vous est possible de discuter dans différents canaux.

Le chat général¶

Par default, lorsque vous discutez dans le chat, vous êtes dans le canal général. Tous les joueurs peuvent voir et vous répondre.

Si jamais vous n’êtes plus dans ce canal, vous pouvez y retourner avec la commande : /g

Le chat local¶

Un peu de tranquillité ça vous dit ? Aucun problème, vous pouvez discuter discrètement ! Seuls les joueurs proches de vous peuvent voir ce que vous racontez dans le chat lorsque que vous êtes dans le canal local.

Pour rejoindre ce canal : /l

Le chat des villes et nations¶

Pour parler avec les personnes uniquement de votre ville il suffit de faire la commande /tc <votre message>

Pour parler avec les personnes uniquement de votre nation il suffit de faire la commande /nc <votre message>

Si tu souhaites rester dans un canal utilise la commande sans le message.

Les messages privés¶

Vous pouvez bien évidemment envoyer des messages privés aux autres joueurs. Pour cela, vous devez executer la commande suivante : /msg <pseudo du destinataire> <votre message>

Pour répondre au dernier message reçu, vous pouvez faire la commande /r <votre message

Il est possible de répondre parler avec une personne dans taper en continue les commandes précedentes.

Pour cela vous devez commencer votre message par le tag @pseudo @Zimoune Bonjour, tu vas bien ?

Pour quitter ce mode et retourner dans votre canal principal, vous devez taper exit
