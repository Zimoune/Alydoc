===========
Le chat
===========
Il vous est possible de discuter dans différents canaux.


*************************
Le chat général
*************************
Par default, lorsque vous discutez dans le chat, vous êtes dans le canal *général*. Tous les joueurs peuvent voir et vous répondre.

Si jamais vous n'êtes plus dans ce canal, vous pouvez y retourner avec la commande : ``/g``

*******************
Le chat local
*******************
Un peu de tranquillité ça vous dit ? Aucun problème, vous pouvez discuter discrètement !
Seuls les joueurs proches de vous peuvent voir ce que vous racontez dans le chat lorsque que vous êtes dans le canal local.

Pour rejoindre ce canal : ``/l``

*******************
Les messages privés
*******************
Vous pouvez bien évidemment envoyer des messages privés aux autres joueurs.
Pour cela, vous devez executer la commande suivante : ``/msg <pseudo du destinataire> <votre message>``
